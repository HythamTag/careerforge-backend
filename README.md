# careerforge-backend

Backend API for the CareerForge ATS SaaS platform.

## Branches

- `main`: protected, release-ready code
- `develop`: integration branch for all feature branches

## Getting Started

```bash
git clone https://github.com/HythamTag/careerforge-backend.git
cd careerforge-backend

cd backend
npm install
cp .env.example .env
npm run dev
```
